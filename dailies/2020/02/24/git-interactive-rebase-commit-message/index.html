Edit git commit message during interactive rebase

February 24, 2020

  • DAILIES
  • GIT (1)
  • REBASE (1)
  • COMMIT (1)

Previously, this is how I would edit git commit messages beyond HEAD within the history.

$ git rebase -i ${HASH}^1
...Mark commits with `edit` command
$ git commit --amend
...Update message
$ git rebase --continue

Thankfully there is a shortcut for this flow as it’s a bit cumbersome.

$ git rebase -i ${HASH}^1
...Mark commits with `reword` command
...Update message

More info in the git docs here.
